# Afusekt-Desktop

欢迎使用 Afusekt 桌面端

桌面端指 MACOS / Windows

对于账号功能：支持云端服务器同步，支持多线路服务同步

目前经过测试桌面的播放核心完全可以解码杜比(作者个人设备支持杜比视界，暂时不清楚不支持的设备会是什么样)

当前版本为测试版本为了减少作者压力目前仅提供 Afusekt 角色为永久PRO 的用户进行测试，不建议大家为了尝鲜或者是想加入测试去购买自己不需要的PRO

目前版本设定为必须登录，如果您不是Afusekt 角色为永久PRO 则无法登录(大家也不用主动去询问是否可以加入)

目前的测试版本 拥有Emby，jellyfin ，115网盘，123网盘(macos)的添加能力

播放能力目前仅支持Emby与jellyfin

目前测试版本并非稳定所以遇到任何问题可以在  <a >TG</a> 反馈，TG有专门的desptop开发者【无网络环境的的也可加此QQ：3013517385】

对应windows/mac 的会员同步说明：桌面端的会员不与其他的端同步(但只要是买过永久和PRO 的都有极大优惠)

对于定价，正式版本出来后将公布
